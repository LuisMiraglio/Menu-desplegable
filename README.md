# Menú Desplegable Premium

Versión evolucionada de un sidebar animado en HTML, CSS y JavaScript con enfoque visual profesional para portfolio.

## Mejoras incluidas
- Nueva paleta **grafito + coral** con estilo sobrio y moderno.
- Sidebar lateral animado con branding, perfil y mejor jerarquía de estados.
- Ítems de menú totalmente funcionales con vistas dinámicas: Home, About, Messages, Photos y Settings.
- Microinteracciones de hover/active y tooltips cuando el menú está colapsado.
- Layout responsive para desktop, tablet y móvil.

## Tecnologías
- HTML5 semántico
- CSS (manteniendo archivo SCSS base)
- JavaScript Vanilla
- Ionicons

## Uso
1. Clona el repositorio.
2. Abre `index.html` en el navegador.
3. Interactúa con el menú lateral y revisa cada sección dinámica.
